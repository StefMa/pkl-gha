  • link
    `package-version-ids`: (String|Number|Boolean)?

    Comma separated string of package version ids to delete.

  • link
    owner: (String|Number|Boolean)?

    Owner of the repo containing the package version to delete. Defaults to the owner of the repo running the action.

  • link
    `package-name`: (String|Number|Boolean)?

    Name of the package containing the version to delete.

  • link
    `package-type`: (String|Number|Boolean)?

    Type of package. Can be one of container, maven, npm, nuget, or rubygems.

  • link
    `num-old-versions-to-delete`: (String|Number|Boolean)?

    Number of versions to delete starting with the oldest version. Defaults to 1.

  • link
    `min-versions-to-keep`: (String|Number|Boolean)?

    Number of versions to keep starting with the latest version. By default keeps no version. To delete all versions set this as 0.

  • link
    `ignore-versions`: (String|Number|Boolean)?

    Regex pattern for package versions to ignore. Defaults to delete all versions.

  • link
    `delete-only-pre-release-versions`: (String|Number|Boolean)?

    Deletes only pre-release versions. The number of pre-release versions to keep can be specified by min-versions-to-keep. When this is set num-old-versions-to-delete and ignore-versions will not be taken into account. By default this is set to false

  • link
    `delete-only-untagged-versions`: (String|Number|Boolean)?

    Deletes only untagged versions in case of a container package. Does not work for other package types. The number of untagged versions to keep can be specified by min-versions-to-keep. When this is set num-old-versions-to-delete will not be taken into account. By default this is set to false

  • link
    token: (String|Number|Boolean)?

    Token with the necessary scopes to delete package versions. If num-old-versions-to-delete is used the token also needs the read packages scope. Defaults to github.token scoped to the repo running the action. To delete package versions of a package outside the repo the action is running in use a Personal Access Token stored as a secret.
